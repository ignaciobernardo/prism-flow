# Prism Flow

Procedural spectral motion rendered with a custom animated WebGL renderer, with editable output sizing, timeline transport, persistence, and image/video export.

**Live:** https://natochi.cv/tools/prism-flow/

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build          # outputs to dist/
npm run verify:final   # typecheck, tests, build, browser tests
```

Built with React, Vite and TypeScript. See `specs/` for the product spec.
